# STA141B-airbnb-project

## Data

The final dataset used in the analysis is located in:

data/processed_data/final.csv

This dataset is constructed by merging:

- Inside Airbnb listing data
- Zillow rent index
- US Census socioeconomic indicators

Here is the link for Census dataset:
https://www.census.gov/data/developers/data-sets.html

Spatial boundary data comes from the US Census TIGER/Line shapefiles.
Download link:
https://www.census.gov/geographies/mapping-files/time-series/geo/tiger-line-file.html
